## Description

[NodeJS]

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# develop
$ npm run dev

# production
$ npm run prod
```

## Deploy with Railway

```bash
# Install Railway CLI
brew install railway

# Authen Railway
railway login

# Connect project
railway link

# Deploy
railway up
```
